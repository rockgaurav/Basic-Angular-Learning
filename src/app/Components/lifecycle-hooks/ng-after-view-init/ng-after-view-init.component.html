ngAfterViewInit
What is an ngAfterViewInit?

ngAfterViewInit is called when the component's template successfully rendered in DOM â€” including the child components and @ViewChild elements.

Use: 1. For Auto Scroll.

2. Required animation after page rendered.

3. Want to check DOM size.

4. Use ngAfterViewInit() for safe DOM access tasks like focusing inputs, initializing plugins, measuring elements, or using child components.

Real Scenario: It focus input box automatically while page loading.

Not Use ngAfterViewInit:
  1. If you try to manipulate the DOM using @ViewChild in ngOnInit(), it may result in an error or undefined because the view/template hasn't been fully rendered yet .
