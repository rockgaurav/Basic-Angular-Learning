ngOnChanges
What is an ngOnChanges?

ngOnChanges When @input property change, it's detect.

Use: Detect the @input changes and fetch new data.

Real Scenario: Detect and fetch data when @input property change from parent component.

Not Use ngOnChanges:
  1. Don't know the new value.
  2. UI will not update.
  3. Sometime logic will work on old data
