ngAfterViewChecked
What is an ngAfterViewChecked?

ngAfterViewChecked when your component's view or child view updates frequently and you want to run some logic after each update.

Use: When UI updated continueously like (dynamic values, repeated changes)

Real Scenario: Imagine you're building a chat application. When a new message is added, you want the chat box to automatically scroll to the bottom so the latest message is visible.

Not Use ngAfterViewChecked:
  1. If UI updates dynamically but you don’t need to run any logic after that, then it's okay — no problem.
  2. Not necessary to use.
